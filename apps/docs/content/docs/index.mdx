# Winner Spinner

Welcome to the Winner Spinner documentation! This site contains everything you need to know about developing, using, and maintaining the Winner Spinner project.

## Project Overview

Winner Spinner is a Chrome extension designed for raffle companies, primarily targeting the UK market. The extension features:

- A customizable spinner for raffle selections
- CSV data import functionality
- Integration with Directus for backend data management
- Comprehensive documentation and marketing website

The project is built with modern web technologies:
- TypeScript for type safety
- React for UI components
- Next.js for the marketing and documentation sites
- Tailwind CSS for styling
- Turborepo for monorepo management
